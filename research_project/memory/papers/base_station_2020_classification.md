## Abstract

 The use of millimeter-wave (mmWave) bands in 5G networks introduce a new set of challenges to network planning. Vulnerability to blockages and high path loss at mmWave frequencies require careful planning of the network to achieve the desired service quality. In this paper, we propose a novel 3D geometry-based framework for deploying mmWave base stations (gNBs) in urban environments by considering first-order reflection effects.
